# KINMEI Business Growth

Static landing page and event detail page for KINMEI Business Growth.

## GitHub Pages

This repository is deployed with GitHub Pages from the `main` branch root.

- Repository: https://github.com/takuronakakyama/kinmei-print-growth
- Site: https://takuronakakyama.github.io/kinmei-print-growth/

The top page is served from `index.html`, the event page is `event.html`, and events are managed in `events.js`.
